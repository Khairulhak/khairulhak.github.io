Body Mass Index (BMI) Calculator

Shiny Application

Khairul Ashraf

Body Mass Index (BMI) Calculator is an app to calculate BMI given 2 inputs, namely weight and height. This simple yet useful app will let user to navigate the input using slider to really have an idea on the any ideal weight and height.

User Interface (ui.R)

  • User interface of this app is really simple and minimalist. It contains the slider for inputting values and main bar to display the calculated result.

  • There will be no risk of error input since this app isn't using any input form field. Thus, there is no input validation needed.

  • Users also will experience smooth and easy navigation since there is not submit button and no need to hit back button to recalculate.

Processing (server.R)

  • For processing part, all calculation is done using R programming in server.R file.

  • BMI is calculated by dividing the weight with the height to the power of 2.

  • BMI is considered underweight is the result is below 18.5, normal if in the interval of 18.5 to 30 and considered overweight if above 30.

Code for Server.R


shinyServer(function(input, output) {

    paste0("For weight ",input$mass, " kg and height ", input$height, " cm.")
  }) 
  output$result <- renderText({
    bmi = input$mass / (input$height / 100)^2
    
    if      (bmi <= 18.5) info = "Underweight."
    else if (bmi <= 25.0) info = "Normal."
    else if (bmi <= 30.0) info = "Overweight."
    
    paste0("For the input, BMI calculated is ", "", round(bmi, 2), "", ".
 This is considered as ", info)
  })
})
